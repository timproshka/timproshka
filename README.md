# timproshka

Personal portfolio site for **Timofey Perminov** — Web Developer & AI Automation Engineer.

**Live:** [timproshka.vercel.app](https://timproshka.vercel.app)

## What I build

- **Fast Astro sites** for B2B service firms — shipped in 10 days, sub-1.5s LCP
- **AI automation** — Telegram/web chatbots, n8n workflows, report pipelines that save 15+ hours a week
- **Custom interactive UI** — Three.js, GSAP, Lenis

## Stack

Astro · React · TypeScript · Tailwind · Three.js · GSAP · Lenis · n8n · Python · Node

## Features

- 3D particle morph hero (28,000 particles, custom shader, ~60 fps on mobile)
- `prefers-reduced-motion` fallback with static heading (full accessibility)
- Mobile-first scroll direction lock (vertical scrolls page, horizontal drives physics)
- LCP optimized — text visible immediately, canvas is progressive enhancement
- Static-first Astro output → instant TTFB
- Custom contact form with Telegram integration

